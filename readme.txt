Условие задачи:
Пришла весна, и дорогу в университет затопило, поэтому студенты передвигаются прыжками по кочкам. Дорога поделена на клетки. На каждой клетке может присутствовать кочка. Дорога считается преодолённой, когда студент попадает на последнюю кочку. Для заданной конфигурации клеток необходимо определить, возможно ли пройти по такой дороге, не обмочив ноги, или нет.
При этом:
1. Первый прыжок всегда на одну клетку.
2. Если предыдущий прыжок был на k клеток, то следующий может быть на k-1, k или k+1 клеток.
3. Прыгать можно только вперёд.
4. На первой клетке всегда есть кочка.
5. На всей дороге будет не меньше двух кочек.

Описание кнопок:
Старт - запускает игру.
+Клетка - добавляет пускую клетку.
-Клетка - удаляет последную клетку.
Отмена - откатывает студента на один прыжок назад.
Слабый прыжок, Средний прыжок, Сильный прыжок - расстояние, на которое может прыгнуть студент.

Белые квадратики(клетки) поначалу пустые. Чтобы появилась кочка(чёрный квадратик), нужно нажать на него. Для удаления кочки с клетки 
повторно кликнуть на неё.
Если первая клетка не кочка, то игру запустить нельзя.

Первый ряд чисел, появляющихся при нажати Старт, Слабый прыжок, Средний прыжок и Сильный прыжок, - это история позиций, на который
был студент. Второй ряд - это сила прыжка, с которой он пригает на следующую клетку.
